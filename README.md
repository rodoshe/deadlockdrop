## Deadlock Drop

A community-driven feature request platform for Valve's game Deadlock. Players can submit ideas, vote on what matters most, and track requests through a public roadmap, built to give the community a voice during the game's development.


## Features

- **Feature Requests** - Submit ideas across categories like Heroes, Mechanics, Ranked, Map, Items, and UI/UX
- **Voting System** — Upvote requests you support; anyone can vote, no account required
- **Comments** — Logged-in users can discuss requests and leave feedback
- **Roadmap** — Public Kanban board showing requests by status: Requested → Planned → Added
- **User Accounts** — Register, log in, and track your votes and comments from your profile
- **Forgot Password** — Email-based password reset flow
- **Admin Panel** — Manage requests, update statuses, edit or delete entries
- **About Page** — Full hero roster with abilities and links to the Deadlock wiki
- **Persistent Database** — PostgreSQL via Supabase, so data never resets

## Tech Stack

- **Backend** — Python, Flask
- **Database** — PostgreSQL (Supabase)
- **Frontend** — HTML, CSS, Jinja2 templating
- **Email** — Flask-Mail with Gmail SMTP
- **Hosting** — Render
- **Version Control** — Git, GitHub
